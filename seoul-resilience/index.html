v0.1 모형 범위

이 버전은 공공 API 연계 전 시각 프로토타입입니다. 행정동·자치구 경계는 정적 GeoJSON을 사용하고, 대비상태는 거리·수요·교통지연·복합기후 스트레스에 기반한 모의 지표로 계산합니다.

v0.2에서는 서울 열린데이터광장, 기상청 APIHub, 공공데이터포털 키를 GitHub Secrets에 저장하고 GitHub Actions가 정적 JSON 스냅샷을 주기적으로 갱신하는 구조로 확장합니다.

경계: Seoul Maps JUSO 2015 모형: static client-side stress test 보안: 공개자료·권역집계 중심
